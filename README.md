Sistema de Biblioteca em Java

Sistema de gerenciamento de biblioteca desenvolvido para aplicar conceitos de Programação Orientada a Objetos.

CONCEITOS APLICADOS
- Herança
- Encapsulamento
- Polimorfismo
- Abstração
- Princípios SOLID
- Separação em pacotes (model, service, repository, exception)

     FUNCIONALIDADES
- Cadastro de livros e revistas
- Cadastro de usuários
- Empréstimo e devolução de itens
- Consulta de disponibilidade
- Histórico de empréstimos por usuário
- Tratamento de exceções personalizado
